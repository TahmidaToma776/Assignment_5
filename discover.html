Question-1: What are the different ways to select an element in the DOM?

ANSWER:

getElementById(id): Selects by ID.
getElementsByClassName(class): Selects by class name.
querySelector(selector): Selects the first matching element using CSS selectors.
querySelectorAll(selector): Selects all matching elements using CSS selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

ANSWER:

  • innerHTML: Retrieves or sets HTML content, including tags.
  • innerText: Retrieves or sets visible text, excluding hidden text.
  • textContent: Retrieves or sets raw text, including text in hidden elements.

Question-3: What is event delegation in the DOM?

ANSWER:

Event delegation involves attaching a single event listener to a parent element to handle events for multiple child elements, improving performance and handling dynamic content.

Question-4: What is event bubbling in the DOM?

ANSWER:

Event bubbling refers to an event starting from the target element and propagating up to its ancestors, triggering event listeners at each level. It can be stopped using event.stopPropagation().
