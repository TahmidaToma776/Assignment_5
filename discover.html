Question-2: What is the difference between innerHTML, innerText, and textContent ?

ANSWER:

innerHTML: sets the HTML content inside an element
innerText: sets the visible text inside an element
textContent: sets all text inside an element, including hidden text
So the main difference between them are:

innerHTML can parses HTML with Hidden Text innerText can Just only reads the visible Text On the other hand, textContent can show the hiddenText

Question-3: What is event delegation in the DOM?

ANSWER:

Event Delegation is an approach in JavaScript where a single event handler is bound to the parent instead of multiple child elements.

Question-4: What is event bubbling in the DOM?

ANSWER:

Event bubbling is a DOM phenomenon in which an event starts from the target element and travels up its ancestors to the root.
